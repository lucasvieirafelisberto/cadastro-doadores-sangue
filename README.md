# 🩸 Cadastro de Doadores de Sangue

## 📌 Descrição
Sistema simples em Python para cadastro e validação de doadores de sangue.

## 🚀 Funcionalidades
- Cadastro de doador
- Cálculo automático de idade
- Validação para doação

## 🧠 Regras
- Peso mínimo: 50kg
- Idade: entre 16 e 69 anos

## ▶️ Como executar
```bash
python main.py
